## SLIDE 8: GAME LOOP - CƠ CHẾ HOẠT ĐỘNG
--------------------------------------------------------------------------------
### GameThread:
┌──────────────────────────────────────┐
│           while (running)            │
│  ┌────────────────────────────────┐  │
│  │  1. update() - Cập nhật logic │  │
│  │  2. draw() - Vẽ lên Canvas    │  │
│  │  3. sleep() - Chờ đủ frame    │  │
│  └────────────────────────────────┘  │
└──────────────────────────────────────┘

### FPS Control:
- Easy mode: 5 FPS (200ms/frame)
- Hard mode: 8 FPS (125ms/frame)
- Tốc độ tăng dần theo điểm số

## SLIDE 9: XỬ LÝ GAME LOGIC
--------------------------------------------------------------------------------
### Snake Movement:
1. Tính vị trí mới của đầu rắn
2. Thêm vị trí mới vào đầu danh sách
3. Xóa segment cuối (nếu không ăn food)

### Collision Detection:
- Va chạm tường: head.x/y ngoài grid
- Va chạm thân: head trùng với body segment
- Ăn food: head.position == food.position

### Food Spawning:
- Random vị trí trên grid
- Kiểm tra không trùng thân rắn
- Respawn sau khi bị ăn

## SLIDE 10: QUẢN LÝ ÂM THANH
--------------------------------------------------------------------------------
### SoundManager (Singleton Pattern):
┌─────────────────────────────────────────┐
│  MediaPlayer        │  SoundPool        │
│  ─────────────────  │  ─────────────   │
│  • Menu music       │  • Eat SFX        │
│  • Game music       │  • Die SFX        │
│  • Loop playback    │  • Short sounds   │
└─────────────────────────────────────────┘

### Luồng xử lý:
- init() → Load tất cả sounds
- playMenuMusic() ↔ playGameMusic()
- playEatSound(), playDieSound()
- pauseAll() khi minimize app

## SLIDE 12: INPUT HANDLING
--------------------------------------------------------------------------------
### GestureDetector + InputHandler:
┌─────────────────────────────────────────┐
│  User swipes on screen                  │
│            ↓                            │
│  onFling(e1, e2, velocityX, velocityY)  │
│            ↓                            │
│  Calculate diffX, diffY                 │
│            ↓                            │
│  |diffX| > |diffY| → Horizontal swipe   │
│  |diffY| > |diffX| → Vertical swipe     │
│            ↓                            │
│  gameEngine.changeDirection(Vector2D)   │
└─────────────────────────────────────────┘

### Threshold Values:
- SWIPE_THRESHOLD = 100 pixels
- SWIPE_VELOCITY_THRESHOLD = 100

## SLIDE 17: Q&A
--------------------------------------------------------------------------------
### Câu hỏi thường gặp:

Q: Tại sao dùng Canvas thay vì Game Engine?
A: Để học cách vẽ đồ họa từ đầu, hiểu rõ game loop

Q: SharedPreferences vs SQLite?
A: Dữ liệu đơn giản, key-value → SharedPreferences phù hợp hơn

Q: Tại sao dùng Singleton cho Manager classes?
A: Đảm bảo 1 instance duy nhất, truy cập global thuận tiện

Q: FPS được tính như thế nào?
A: targetTime = 1000 / targetFPS, sleep nếu frame xử lý nhanh hơn
